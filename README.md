# SPIE
SPIE (Sistem perbankan dan Investasi yang ter-Enskripsi) adalah projek c++ terakhir masa smp dan merupakan projek ke 3. Memiliki 2 mode utama yaitu sistem perbankan dan investasi,disini Sistem Perbankan menawarkan 9 fitur utama yaitu:
1.login akun 
2.bikin akun 
3.tranfer 
4.tarik tunai 
5.ganti informasi akun 
6.lihat riwayat 
7.pembelian 
8.hutang piutang / pinjaman berbunga 
9.lihat informasi akun 
lalu kemudian untuk mode investasi menawarkan jenis 4 investasi yaitu:
1.emas 
2.obligasi dari perusahaan / pemerintah
3.saham
4.crypto
dan masing masing investasi memiliki konsekuensi instan tanpa menunggu waktu.
